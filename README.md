> # 關關難過關關過的輔助系統 #


目標
====

輔助使用者在面對這樣子的流程時，可以紀錄 / 檢視目前進度：

```
Start ---> A    ---> B  ---> ... ---> X    ---> End
           + A1                       + X1
           + A2                       + X2
                                      + X3
```

* A、B、...、X 稱之為「檢查點」。
* 檢查點之間是有前後關係的，亦即 A 必須確認完畢才能開始進入 B。
* 檢查點下可以有「檢查細項」（如 A 下有 A1、A2）。
	檢查細項都確認後，該檢查點才可以進行確認。

本系統另一個重點功能為：檢查點與檢查細項都可由使用者自行定義。
